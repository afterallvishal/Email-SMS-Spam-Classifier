# Spam Classifier

A Machine Learning based Spam Message Classifier using Natural Language Processing.

## About Project

This project predicts whether a given SMS/Email message is Spam or Not Spam using Machine Learning techniques.

## Technologies Used

- Python
- NLP
- Scikit-learn
- TF-IDF Vectorizer
- Streamlit

## Features

- Spam and Ham message classification
- Text preprocessing using NLP techniques
- User-friendly Streamlit web interface
